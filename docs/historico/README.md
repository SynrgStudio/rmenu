# Histórico

Documentación histórica y reportes previos del desarrollo de `rmenu`.

Estos archivos se conservan como referencia, pero no son la fuente oficial actual de specs o arquitectura. Las specs vigentes viven en el root del repositorio.

Contenido:

- `private-docs/`: planes, snapshots y reportes internos antiguos.
- `modulesREPORT/`: documentación y ejemplos históricos del trabajo de módulos.
- `modulesREPORTE/`: reportes históricos adicionales.
